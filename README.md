# Public Art Watcher Feed

This repository contains the public static report files for the GrantCraft Public Art Watcher.

The main watcher runs in a private repository. After each update, selected public report files are copied here so they can be displayed on GrantCraft.art without exposing the private watcher code, credentials, or internal maintenance files.

## Published files

* `weekly_report_fragment.html` — the embeddable report fragment used on the GrantCraft.art Public Art Watcher page.
* `weekly_report.html` — the full HyperText Markup Language version of the report.
* `weekly_report.txt` — the plain text version of the report.

## Public page

The report is displayed at:

https://grantcraft.art/public-art-watcher/

## Source

This repository is only a public feed. The working source repository remains private.
